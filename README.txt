Daelijek Parameters Retrieval

Description:
The "Daelijek Parameters Retrieval" program is a Python application designed to retrieve various system parameters and information using a graphical user interface (GUI).

Requirements:

	Python 3.x
	Tkinter (Python's standard GUI toolkit)
	psutil (Python package for system monitoring)
	Platform-specific dependencies (depending on the system information being retrieved)
	For running the executable (exe) file:
	No additional dependencies are required.

Usage:

1.Install Dependencies:
Ensure you have Python installed on your system. You can download it from the official Python website: python.org. Tkinter and psutil are included in the standard library, so no additional installation is required.

2.Clone the Repository:
Clone or download the "Daelijek Parameters Retrieval" repository to your local machine.

3.Navigate to the Directory:
Open a terminal or command prompt and navigate to the directory where the program files are located.

4.Run the Program:
Execute the Python script os_parameters.py by running the following command:
python os_parameters.py
This will launch the GUI application.

5. Running from the Executable (exe) File:
Download the Executable: If provided, download the executable (exe) file of the program to your local machine.

Run the Executable: Double-click on the executable file (os_parameters.exe) to run the program.

Retrieve System Parameters: Once the GUI application is launched, you'll see a list of options representing different system parameters. Click on any option to retrieve the corresponding information. The retrieved information will be displayed in a messagebox.

Exit the Program: You can close the GUI application window to exit the program.
